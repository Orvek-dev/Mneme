# V2 Complete Public Package MVP

## Goal

Make Mneme v2 understandable and reproducible as a public team-agent memory
layer, without adding hosted SaaS surfaces or private workspace artifacts.

## Scope

- v2 quickstart for a complete local flow;
- team-agent workflow documentation;
- security model documentation;
- evaluation documentation;
- public-safe team-agent operations example;
- MCP-style agent configuration example;
- static public-safe demo report;
- quality-gate coverage for the demo.

## Completion Criteria

- a new user can run one script and inspect a v2 handoff package;
- the script proves private memory redaction, quarantine omission, checksum
  verification, quality findings, and project access denial;
- docs explain what v2 does and does not claim;
- the full repository quality gate runs the public example;
- no personal paths, secrets, or private templates are committed.
